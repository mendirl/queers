  • il y a des balises de 3 couleurs distinctes :
    • rouge ou violet : centre d'intérêt
    • vert : stations se trouvant dans un rayon définit
    • bleu : stations étant loin de tout
  • les urls peuvent avoir des paramètres :
    • lat : lattitude du point sur lequel on veut se center (décimal)
    • lng : longitude du point sur lequel on veut se center (décimal)
    • rad : le rayon du disque (km), par défaut à 0.4
    • place : type de données que l'on veut afficher (park, museum, ...)
    • all : affichage des stations bleues (booléen)
